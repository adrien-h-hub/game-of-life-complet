ENGLISH VERSION
---------------

Welcome to Conway's Game of Life Enhanced Edition!

FEATURES:
• 8 Beautiful color themes (Neon, Ocean, Forest, Sunset, etc.)
• Time travel: Rewind and forward through generation history
• Auto-pause on stable patterns (oscillators, still lifes)
• Pattern detection system
• Random 3×3 pattern generator (4-9 cells)
• Real-time statistics and mini-map
• Interactive drawing and zooming
• Cross-platform support

SYSTEM REQUIREMENTS:
• Operating System: Windows 10/11, Linux, or macOS 10.15+
• RAM: 512 MB minimum
• Processor: x64 compatible
• Graphics: OpenGL 3.3+ support
• Disk Space: 50 MB

INSTALLATION STEPS:
1. Click "Next" to continue
2. Choose installation directory (default recommended)
3. Select if you want a desktop shortcut
4. Click "Install" to begin installation
5. Wait for installation to complete
6. Click "Finish" to launch the application

AFTER INSTALLATION:
• Launch from Start Menu or desktop shortcut
• Press H in the application for help
• Press R to generate a random pattern
• Press SPACE to start simulation
• Press K to change themes

For more information, visit:
https://github.com/yourusername/GameOfLife

TROUBLESHOOTING / DÉPANNAGE
----------------------------

ENGLISH:
Q: The application doesn't start?
A: Make sure you have OpenGL 3.3+ drivers installed. Update your graphics drivers.

Q: Black screen or no window appears?
A: Check Windows Task Manager. The app might be running in the background.
   Try Alt+Tab to find the window.

Q: Performance issues?
A: Press +/- to adjust simulation speed. Close other applications.

FRANÇAIS :
Q : L'application ne démarre pas ?
R : Assurez-vous d'avoir les pilotes OpenGL 3.3+ installés. Mettez à jour vos pilotes graphiques.

Q : Écran noir ou pas de fenêtre ?
R : Vérifiez le Gestionnaire des tâches Windows. L'app peut tourner en arrière-plan.
   Essayez Alt+Tab pour trouver la fenêtre.

Q : Problèmes de performance ?
R : Appuyez sur +/- pour ajuster la vitesse. Fermez les autres applications.
